Sides
  • constrain(sides:_:to:constant:isSafe:)

    Adds and activates one or more side constraints.

    Declaration

    Swift

    @discardableResult
public func constrain(sides: Set<Side>, _ relation: Relation = .equal, to view: UIView, constant: CGFloat = 0, isSafe: Bool = false) -> Sides

    Parameters

    sides

    One or more Side that correspond to the calling view.
    relation

    The relationship between the left side of the constraint and the right side of the constraint. Defaults to .equal
    view

    The view to pin the calling view to.
    constant

    The constant offset for the constraint. A positive value moves the given sides towards the center of the given view. Defaults to 0
    isSafe

    if true the safeAreaLayoutGuide for the given view is used. Defaults to false

    Return Value

    A new Sides instance that contains the appropriate constraints.

Sides Y
  • constrain(side:_:to:of:constant:isSafe:)

    Adds and activates a SideY (top/bottom) constraint that will be pinned to a SideY of the given view.

    The main purpose of this function is for both sides to be different. Example:

    view.constrain(side: .top, to: .bottom, of: view2)

    OR

    view.constrain(side: .bottom, to: .top, of: view2)
