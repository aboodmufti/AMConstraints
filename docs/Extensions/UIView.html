Sides
  • constrain(sides:_:to:constant:isSafe:priority:)

    Adds and activates one or more side constraints.

    Declaration

    Swift

    @discardableResult
public func constrain(sides: Set<Side>, _ relation: Relation = .equal, to view: UIView, constant: CGFloat = 0, isSafe: Bool = false, priority: UILayoutPriority = .required) -> Sides

    Parameters

    sides

    One or more Side that correspond to the calling view.
    relation

    The relationship between the left side of the constraint and the right side of the constraint. Defaults to .equal
    view

    The view to pin the calling view to.
    constant

    The constant offset for the constraint. A positive value moves the given sides towards the center of the given view. Defaults to 0
    isSafe

    if true the safeAreaLayoutGuide for the given view is used. Defaults to false
    priority

    The priority of the constraint. Defaults to .required

    Return Value

    A new Sides instance that contains the appropriate constraints.

Vertical Sides
  • constrain(side:_:to:of:constant:isSafe:priority:)

    Adds and activates a SideY (top/bottom) constraint that will be pinned to a SideY of the given view.

    The main purpose of this function is for both sides to be different. Example:

    view.constrain(side: .top, to: .bottom, of: view2)

    OR

    view.constrain(side: .bottom, to: .top, of: view2)

Miscellaneous
  • constrain(side:view:at:priority:)

    constrains the given side of the view to the given view’s normalized position.

    Top:

            ___________ 0
        |         |
 top    |         |
 ------ |  View   |- 0.5 <-- normalizedPosition
 |    | |         |
 |self| |_________| 1

    Bottom:

            ___________ 0
 |    | |         |
 |self| |         |
 ------ |  View   |- 0.5 <-- normalizedPosition
 bottom |         |
        |_________| 1

    Left:

           _____
        self| right
       _____|
        ___________
        |         |
        |  View   |
        |_________|
        0    |    1
            0.5 <-- normalizedPosition

    Right:

                 _____
       left |self
            |_____
        ___________
        |         |
        |  View   |
        |_________|
        0    |    1
            0.5 <-- normalizedPosition

    Declaration

    Swift

    public func constrain(side: Side, view: UIView, at normalizedPosition: CGFloat, priority: UILayoutPriority = .required) -> NSLayoutConstraint

    Parameters

    view

    The view to which the calling view should be constrained.
    normalizedPosition

    vertical/horizontal position between 0 (top) and 1 (bottom).
    priority

    The priority of the constraint. Defaults to .required
